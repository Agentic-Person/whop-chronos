# Video Components

This directory contains components related to video functionality:

- Video upload interface
- Video player with progress tracking
- Video list/grid displays
- Video metadata editing
- Video processing status indicators

## Planned Components

- `VideoUploader.tsx` - Drag-and-drop video upload with progress
- `VideoPlayer.tsx` - Custom video player with AI chat integration
- `VideoList.tsx` - List view of videos with filtering
- `VideoCard.tsx` - Individual video card component
- `VideoMetadata.tsx` - Edit video title, description, etc.
- `ProcessingStatus.tsx` - Shows transcription/embedding progress
