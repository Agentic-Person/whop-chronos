# Course Components

This directory contains components for course management:

- Course builder interface
- Module organization
- Drag-and-drop course structure
- Course preview
- Student course view

## Planned Components

- `CourseBuilder.tsx` - Main course creation interface
- `ModuleEditor.tsx` - Edit individual modules
- `VideoAssignment.tsx` - Assign videos to modules
- `CoursePreview.tsx` - Preview how students see the course
- `CourseCard.tsx` - Course card for listings
- `ModuleList.tsx` - List of modules with reordering
