# Chat Components

This directory contains components for the AI chat interface:

- Chat message display
- Chat input with AI suggestions
- Video timestamp citations
- Chat history management
- RAG search results display

## Planned Components

- `ChatInterface.tsx` - Main chat container
- `ChatMessage.tsx` - Individual message component
- `ChatInput.tsx` - Message input with send button
- `VideoReference.tsx` - Clickable video timestamp citations
- `ChatHistory.tsx` - List of previous chat sessions
- `SuggestedQuestions.tsx` - AI-suggested question prompts
