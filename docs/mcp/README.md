# MCP Documentation

This directory contains all documentation related to Model Context Protocol (MCP) servers used in the Chronos project.

## Documentation Files

### [MCP_CONFIGURATIONS.md](./MCP_CONFIGURATIONS.md)
Complete guide to MCP configurations for Chronos, including:
- Main configuration (`.mcp.json`) for Whop API, Whop Docs, and Supabase
- UI configuration (`ui.mcp.json`) for browser testing
- Understanding the difference between Whop API server and Whop Docs server
- Switching between configurations
- Common workflows and best practices

### [WHOP_MCP_SETUP.md](./WHOP_MCP_SETUP.md)
Step-by-step guide for setting up the Whop MCP server on a new computer:
- Installation instructions
- Configuration for Windows, macOS, and Linux
- Environment variable setup
- Troubleshooting common issues
- Testing and verification

### [UI_MCP_GUIDE.md](./UI_MCP_GUIDE.md)
Guide for using UI-focused MCP servers:
- Puppeteer for browser automation
- Playwright for multi-browser testing
- Sequential Thinking for architecture planning
- Fetch for documentation retrieval
- Memory for pattern persistence

## Quick Start

1. **Set up Whop MCP Server**: Follow [WHOP_MCP_SETUP.md](./WHOP_MCP_SETUP.md)
2. **Configure MCP**: See [MCP_CONFIGURATIONS.md](./MCP_CONFIGURATIONS.md)
3. **Use UI MCP Tools**: Read [UI_MCP_GUIDE.md](./UI_MCP_GUIDE.md)

## Related Files

- `.mcp.json` - Main MCP configuration (project root)
- `.mcp.json.example` - Template configuration (project root)
- `ui.mcp.json` - UI testing configuration (project root)
