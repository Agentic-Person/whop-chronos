# Landing Page Demo Data

This directory contains the transcript and processed data for the landing page interactive demo.

## Files

- `demo-video-transcript.txt` - Raw transcript of the demo video (YouTube: vMZHiBhr0SM)
- `processed-chunks.json` - Processed transcript chunks with embeddings (auto-generated)
- `chapters.json` - Chapter timestamps and summaries (auto-generated)

## Setup Instructions

1. Place your video transcript in `demo-video-transcript.txt`
2. Run `npm run process-landing-transcript` to generate embeddings
3. The interactive chat will automatically use the processed data
